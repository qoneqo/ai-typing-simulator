# Changelog

All notable changes to the **AI Typing Simulator** extension will be documented in this file.

## [0.0.1] - 2026-06-22

### Added
- Initial release of the AI Typing Simulator extension.
- Added commands for running (`ai-typing-simulator.runScript`) and stopping (`ai-typing-simulator.stopScript`) typing simulations.
- Implemented humanized typing delays with natural pauses at punctuation marks, spaces, and after newlines.
- Implemented typing trace commands: `write`, `move` (absolute/relative), `wait`, and `backspace`.
- Supports loading simulator scripts from Clipboard or from a JSON file.
